# 日语 N5 AI 辅导助手（大家的日语专属版）PRD 产品需求文档

## 一、文档概述

### 1.1 产品名称

日语 N5 AI 辅导助手（大家的日语专属版）

### 1.2 产品定位

专为日语 N5 级别、使用《大家的日语》教材学习者打造的 AI 智能辅导工具，无需本地存储教材内容，依托大模型 AI 实时生成单词、语法、课文、听力、例句等学习内容，支持学习进度跟踪、薄弱点标记与个性化复习，实现轻量化、高效化、互动化 N5 备考学习。

### 1.3 目标用户

备考日语 N5、学习《大家的日语》教材的零基础 / 入门级日语学习者，需求为轻量化学习、精准考点练习、进度跟踪、AI 互动答疑。

### 1.4 核心价值

- 无需整理教材资料，AI 实时调取《大家的日语》N5 全内容
- 精准匹配课次 + 知识点，定向练习单词、语法、课文、听力
- 自动记录学习进度与掌握情况，薄弱点反复强化
- 实时 AI 互动，支持指令式学习、答疑、出题

### 1.5 运行平台

移动端（手机优先，适配安卓 / iOS）

## 三、详细功能需求

### 3.1 全局通用模块（所有页面通用）

#### 3.1.1 顶部状态栏

- **产品名称**：日语 N5 AI 辅导助手
- **学习进度展示**：当前学习课次（如第 2 课）、单词 / 语法 / 课文 / 听力熟练度百分比
- **快捷按钮**：一键错题复习、今日学习时长

#### 3.1.2 课次选择栏（固定顶部）

- **单课选择**：第 1 课、第 2 课…… 第 N 课（适配《大家的日语》N5 全课次）
- **范围选择**：单课、1~3 课、1~5 课、全部已学
- **知识点类型快捷切换**：单词 / 语法 / 课文 / 例句 / 听力

#### 3.1.3 底部 AI 交互栏（固定全局）

- **输入框**：支持文字 / 语音输入，提示文案："跟 AI 老师说：练第 2 课语法 / 出 5 道单词题…"
- **发送按钮**：指令提交后，AI 实时生成对应内容
- **支持指令示例**：
  - 练第 X 课单词 / 语法 / 课文
  - 着重练第 X 课第 X 条语法
  - 出 10 道翻译 / 选择题
  - 只复习不会的内容
  - 朗读课文 / 单词

#### 3.1.4 掌握状态标记（所有学习内容通用）

- **三个按钮**：
  - ✅ **会了** — 自动归档，不再优先推送
  - ⚠️ **模糊** — 加入循环复习
  - ❌ **不会** — 加入薄弱本，高频推送
- 所有标记自动同步，AI 记忆学习状态

---

### 3.2 核心学习模块

#### 3.2.1 单词学习模块

- **内容展示**（AI 实时生成）：日语单词 + 读音（平假名 / 罗马音）+ 中文翻译 + N5 极简例句
- **学习模式**：查看学习、默写模式、选择模式、翻译模式、听写模式
- **快捷功能**：只刷不会的单词、批量标记、一键翻译

#### 3.2.2 语法精讲模块

- AI 自动列出当前课次 N5 核心语法点（编号 + 名称）
- **语法详情**：含义 + 接续方式 + 标准例句 + 易错点提示
- **互动功能**：练当前语法、AI 出练习题（选择 / 翻译 / 填空）、语法答疑

#### 3.2.3 课文学习模块

- **AI 实时输出**：课文原文 + 分句对照翻译 + 重点词汇 / 语法标注
- **功能按钮**：AI 朗读、分句跟读、课文填空练习、段落翻译、课文提问

#### 3.2.4 例句练习模块

- 按当前课次 + 知识点生成 N5 标准例句
- **支持互动**：例句翻译、扩写例句、仿写练习、随机刷新例句

#### 3.2.5 听力训练模块

- **AI 生成听力内容**：单词听力、课文短句听力、简短对话听力
- **练习模式**：听选答案、听写填空、跟读打分
- **功能**：循环播放、暂停、语速调节

---

### 3.3 辅助管理模块

#### 3.3.1 薄弱 / 错题本

- **自动收录**：标记"模糊 / 不会"的单词、语法、错题
- **展示**：内容 + 错误原因 + 掌握状态
- **功能**：专项复习、重新测试、批量标记已掌握、清空记录

#### 3.3.2 学习记录

- **进度统计**：已学课次、各模块掌握数量、总学习时长
- **薄弱项统计**：高频薄弱知识点汇总
- **功能**：手动修改学习进度、导出学习记录、重置学习数据

## 四、界面设计需求（移动端）

### 4.1 整体布局

| 区域 | 内容 |
| --- | --- |
| **左侧导航**（抽屉式） | 单词、语法、课文、例句、听力、薄弱本、学习记录 |
| **右侧主内容区** | 课次选择栏 + 学习内容展示区 |
| **顶部** | 全局状态栏 |
| **底部** | AI 交互输入栏 |

### 4.2 视觉风格

- **主色调**：浅蓝色（护眼、专注学习）
- **字体**：清晰无衬线字体，日语 / 中文字号区分合理，对照清晰
- **交互**：点击操作，无复杂层级，一屏一核心任务
- **排版**：简洁留白，重点内容突出，无冗余信息

### 4.3 交互逻辑

1. 点击左侧模块 → 右侧切换对应内容
2. 选择课次 / 范围 → AI 实时刷新对应内容
3. 标记掌握状态 → 自动同步至薄弱本
4. 输入 AI 指令 → 1 秒内加载对应内容
5. 所有操作无跳转，页面内刷新，降低学习成本

## 五、AI 能力需求

| 能力 | 说明 |
| --- | --- |
| **教材知识储备** | 完整掌握《大家的日语》N5 级别所有单词、语法、课文、例句、听力内容 |
| **实时生成能力** | 实时生成练习题、例句、翻译、朗读音频、课文填空内容 |
| **记忆能力** | 永久记忆用户学习课次、掌握状态、薄弱点、学习进度 |
| **互动理解能力** | 精准识别口语 / 文字学习指令，响应学习需求 |
| **适配 N5 考点** | 所有内容严格贴合日语 N5 考试难度，不超纲 |

## 六、非功能需求

| 指标 | 要求 |
| --- | --- |
| **加载速度** | AI 内容响应 ≤ 2 秒 |

| **易用性** | 零基础用户 5 分钟上手，无需教程 |
| **轻量化** | 无需下载教材包，占用内存小 |

## 七、产品流程

```
打开 APP
  → 进入默认课次（上次学习课次）
  → 选择课次 / 范围 / 学习模块
  → AI 加载对应内容
  → 学习内容 → 标记掌握状态 → 同步至薄弱本
  → 底部输入 AI 指令 → AI 定向生成内容 / 答疑 / 出题
  → 退出 APP → 自动保存学习进度与状态
  → 再次打开 → 一键复习薄弱内容 / 继续上次学习
```
